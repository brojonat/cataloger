About

Cataloger uses LLM agents to explore databases and generate comprehensive HTML catalogs. Each catalog includes:

  • Schema Analysis - Table structures, column types, nullability
  • Data Profiling - Row counts, null rates, distributions
  • Temporal Trends - Changes over time, anomalies
  • Executable Scripts - Python code that generated each catalog
